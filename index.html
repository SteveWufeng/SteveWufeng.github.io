How I build with AI

I orchestrate multi-agent teams using OpenCode Ensemble — each agent works in an isolated sandboxed worktree so no agent can touch another's code, and every change goes through human review before it's merged. AI accelerates the process, but people stay accountable. This is how I think we build secure, high-trust software in an AI-assisted world.
